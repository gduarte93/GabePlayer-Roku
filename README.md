## Custom Video Player For Roku

Sideload /out/roku-deploy.zip onto devide, or zip/compress manually

Features
- DASH and HLS support
- Custom Themeable UI
- Skip Ahead, Skip Behind
- Fastforward and Rewing 1x, 2x, 4x speeds
- Title & Description
- Current and Ending duration timestamps
- Closed Caption support
- Custom trick play / seek support for thumbnails when available, and just timestamp when not
- Picture in Picture mode
- Autoplay next video in playlist
